Quiz Web Application 🎯
📌 Project Overview

This project is a simple Quiz Web Application 
The application allows users to answer multiple-choice questions and view their score at the end of the quiz.

The project demonstrates the use of HTML, CSS, and JavaScript to build an interactive and user-friendly web application.

🚀 Features

Multiple choice quiz questions

Interactive user interface

Score calculation at the end of the quiz

Responsive and simple design

Instant feedback for selected answers

🛠️ Technologies Used

HTML5

CSS3

JavaScript

📂 Project Structure

quiz-project
│
├── index.html

└── README.md

▶️ How to Run the Project

Clone the repository

Open the project folder

Run index.html in your web browser

Or you can simply download the ZIP file and open the HTML file.
